 # Restaurant Menu

This project is a Restaurant Menu UI/UX design created as part of my internship task. The goal of this project is to design a visually appealing and user-friendly menu interface that makes it easy for users to explore food items and place orders efficiently.

🎯 Objective

The main objective of this project is to:

Create an attractive and modern restaurant menu interface
Improve user experience through clear navigation
Make food selection simple and engaging
✨ Features
📱 Clean and modern UI design
🍕 Categorized menu (Pizza, Burgers, Desserts, Drinks)
⭐ Ratings and pricing display
🛒 Add to Cart button for each item
🔍 Search bar for easy navigation
👑 Premium Royal Blue & Gold theme
